# gpu path tracer

[![](http://img.youtube.com/vi/6zpkfS8ZBzE/0.jpg)](https://www.youtube.com/watch?v=6zpkfS8ZBzE)

__Controls:__
* mouse movement: look around
* WS: forward and backward
* AD: left and right
* left shift, space: down and up

__Dependencies:__
* OpenGL 4.3
* glew
* glfw3
* glm
* C++11 compiler
