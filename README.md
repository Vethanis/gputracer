# gpu path tracer

__[Video](https://www.youtube.com/watch?v=NJnvfAHZyT4&feature=youtu.be)__

__Controls:__
* mouse movement: look around
* left click: reset image
* WS: forward and backward
* AD: left and right
* left shift, space: down and up


__Dependencies:__
* OpenGL 4.3
* glew
* glfw3
* glm
* clang
